Este site constitui um trabalho realizado pelos estudantes do ifnmg de Montes Claros: Gabriel Vinícius Borges Ferreira, Felipe Silva Santana e João Duarte Colares. Ele tem por objetivo o desenvolvimento de um site que explique a definição de algumas tags html específicas. Para a criação e estilização deste site foi usada a linguagem de marcação de hipertexto html(Hypertext Markup Language) e a linguagem de estilo css(Cascading Style Sheets).
